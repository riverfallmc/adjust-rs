# adjust
``adjust`` - это библиотека/набор утилит для\
написания более чётко структурированного кода,\
которую мы используем для написания своих
сервисов на языке [Rust](https://rust-lang.org)

# Установка
```toml
[dependencies]
adjust= "*"
```
